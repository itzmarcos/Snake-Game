# SNAKE GAME

Jogo da cobrinha desenvolvido para ampliar as praticas em relação a programação em Python.




### Requisitos

 - Ter algum aplicativo que rode o codigo tipo VSCODE ou PYCHARM
 - Baixar as bibliotecas:
   
  Pygame:
  ```
  pip install pygame
```
  
  Random: 
  ```
  pip install random
```
